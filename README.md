Vida Ativa

Visão do Sistema

O Vida Ativa é um aplicativo feito para ajudar as pessoas a terem uma vida mais saudável de forma simples.

Com ele, o usuário pode:

1.Colocar seus dados (nome, idade, peso, altura, etc.)
2.Escolher seu objetivo (emagrecer, ganhar massa, melhorar a saúde)
3.Ver treinos separados por tipo
4.Ver sugestões de alimentação
5.Acompanhar seu progresso

Os dados do usuário podem ser salvos usando banco de dados, permitindo guardar informações mesmo depois de fechar o app.

Objetivo do Projeto

Criar um aplicativo simples que ajude as pessoas a se exercitarem e se alimentarem melhor, tudo em um só lugar.

Funcionalidades

- Cadastro de dados do usuário
- Treinos (cardio, musculação, etc.)
- Treinos completos (HIIT, funcional, boxe, etc.)
- Sugestões de alimentação
- Tela de resultados com progresso
- Salvamento de dados no banco


Tecnologias Utilizadas

- React Native
- JavaScript
- Context API
- React Navigation
- Banco de dados
